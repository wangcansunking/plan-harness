Claude Code Plugin — Structured Project Planning with Agent Teams
11 skills • 13 MCP tools • 6 agent roles • 7 document types
v1.3.2 Interactive HTML Dark/Light Theme

Overview

plan-harness turns the spec/plan phase of a project into a repeatable, high-quality process. It generates interconnected HTML documents through specialized agent teams, with persistent project contexts and iterative review workflows.

Plugin Architecture

Two-Level Context System

Project contexts persist across scenarios. Create once, reuse for every feature.

Full Workflow

Phase 1: Context Creation

Run once per project. Guided conversation captures everything code analysis can't discover.

Phase 2: Document Generation

Each document type is generated by a different combination of agents.

Phase 3: Review Cycle
